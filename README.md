# Unityでプロジェクトを開く
リポジトリをcloneまたはgitに慣れていない場合はzipでダウンロードしてください。<br>
<img width="483" alt="hpg_downloadproject" src="https://user-images.githubusercontent.com/680194/46682486-a394db80-cc28-11e8-8789-c87635b2cae6.png">

ダウンロードしたzipを展開します。<br>
<img width="300" alt="hpg_unzipproject" src="https://user-images.githubusercontent.com/680194/46682603-f7072980-cc28-11e8-9827-c1c95beebc59.png">

次にUnityを起動し、Openから先ほど展開したプロジェクトを選択します。<br>
<img width="1001" alt="hpg_openproject" src="https://user-images.githubusercontent.com/680194/46682656-1bfb9c80-cc29-11e8-9ff6-3756714d8ad0.png"><br>
<img width="423" alt="hpg_selectproject" src="https://user-images.githubusercontent.com/680194/46682663-23bb4100-cc29-11e8-8573-dbe7e0fae46b.png">

アセットのImportが走るので少しまちます。<br>
<img width="426" alt="hpg_assetsprogress" src="https://user-images.githubusercontent.com/680194/46682676-2f0e6c80-cc29-11e8-9fbf-416b2417e6c4.png">

プロジェクトを開くことができました。<br>
<img width="1267" alt="hpg_projectopened" src="https://user-images.githubusercontent.com/680194/46682701-39306b00-cc29-11e8-924c-1c7ab696f18c.png">

そのままでもよいですが、横長ゲームのオススメの画面レイアウトは2by3です。<br>
メニューバーのWindow->Layouts->2by3で設定できます。<br>
<img width="904" alt="hpg_layout2by3" src="https://user-images.githubusercontent.com/680194/46682718-48171d80-cc29-11e8-9d34-361c9c776ab0.png">

ProjectタブのAssets/Scenesフォルダを選択し、HelloProgrammingというシーンファイルをダブルクリックします。<br>
<img width="227" alt="hpg_openscene" src="https://user-images.githubusercontent.com/680194/46682733-5402df80-cc29-11e8-9c77-7822544eb813.png">

ゲームシーンを開くことができました。再生してみましょう。<br>
<img width="1273" alt="hpg_startgame" src="https://user-images.githubusercontent.com/680194/46682737-5e24de00-cc29-11e8-954b-4675dc18513b.png">
